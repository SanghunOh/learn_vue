#### with_vue
+ official nodejs : https://nodejs.org/
+ official npmjs : https://www.npmjs.com/
+ official vuejs : https://vuejs.org/. 
### first time
```
$ node -v. 
$ npm -v
$ npm install -g @vue/cli
$ vue --version
$ vue create with_vue
$ npm run serve

$ npm install axios
```
### second times
```
$ npm install
```
